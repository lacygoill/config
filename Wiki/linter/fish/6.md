Msg: Prefer `-eq`-like operators over `=` ones to compare numeric operands.

# Bad:

    if test "$status_generation" = 0
                                 ^

# Good:

    if test "$status_generation" -eq 0
                                 ^^^

# Rationale: `=` is meant for text *strings*, and not *numbers*.

# Regex:

    \<test\>\s.*[<>=]\s*-\=\d
