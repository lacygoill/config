Msg: Use the `%u` specifier for your user name.

# Bad:

    ExecStart=echo 'my name is john'
                               ^--^

    ExecStart=echo 'my name is $USER'
                               ^---^

# Good:

    ExecStart=echo 'my name is %u'
                               ^^

For more info: `man systemd.unit /SPECIFIERS/;/%u`.

# Regex:
```vim
    '\<' .. $USER .. '\>'
    .. '\|' .. '$USER\>'
```
